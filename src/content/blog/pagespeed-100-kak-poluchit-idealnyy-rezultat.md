## Почему скорость сайта важна

Скорость — это не просто красивая цифра. Она напрямую влияет на ваш бизнес:

- **53% посетителей** уходят, если сайт загружается более 3 секунд
- Google использует скорость как **фактор ранжирования** — быстрые сайты оказываются выше
- Быстрый сайт означает **больше конверсий** — каждая дополнительная секунда снижает конверсию на 7%

## Какие техники я использовал

### 1. Ноль внешнего CSS — всё встроено

CSS сайта включён непосредственно в HTML, а не в отдельный файл. Это исключает дополнительный HTTP-запрос, блокирующий отображение страницы. Результат: страница отображается мгновенно, без ожидания загрузки CSS-файла.

### 2. Оптимизированные изображения WebP

Все изображения в формате WebP, сжатые и масштабированные точно до размера отображения. По сравнению с JPEG, WebP экономит 70-80% размера при том же визуальном качестве.

### 3. Самостоятельно размещённые шрифты

Шрифты размещены на том же сервере, а не загружаются из Google Fonts. Ноль внешних подключений — ноль задержек DNS и TLS.

### 4. Семантический и доступный HTML

Каждый элемент имеет правильную роль: теги `h1`-`h6` в правильном порядке, атрибуты `alt` на изображениях, `aria-label` на кнопках, достаточная контрастность цветов для удобочитаемости.

### 5. Структурированные данные Schema.org

Сайт включает структурированные данные (JSON-LD), которые помогают Google понять, какие услуги я предлагаю, где нахожусь и как со мной связаться.

### 6. Статический сайт — ноль лишнего JavaScript

Сайт создан на Astro — фреймворке, генерирующем статический HTML. Никаких тяжёлых JavaScript-фреймворков — только HTML, CSS и минимально необходимый JS.
